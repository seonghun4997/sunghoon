# 배포 관문 — 배포·크론·외부 연동 시 필독

## 완료 = 스모크 통과
- 배포 후 스모크 통과 로그 없이 "완료" 선언 금지.
- 실행 로그(빌드 통과·HTTP 200·DB 행 반환 등)가 있을 때만 ✅.
- 스모크 관문이 설치된 repo부터 적용 (현재: adscout).

## 수동 조작 문구 금지
- 사용자 화면에 "수동 ○○ 필요" 문구 금지.
- 수동 조작이 필요하면 자동화하거나, 오너에게 보고 후 대장에 🔲로 기록. UI에는 노출하지 않는다.

## 검증 계정
- 스모크 검증은 `test_검증봇` 계정으로 수행한다 (→ `laws/검증.md` 참조).
